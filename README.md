# Sachin Jangir - Portfolio
A modern, responsive, and interactive personal portfolio web application built meticulously with React and Tailwind CSS.

## 🚀 Tech Stack
- **Frontend Framework:** React.js (Vite)
- **Styling:** Tailwind CSS (Utility-first) + Vanilla CSS Variables (Theming)
- **Animations:** Framer Motion
- **Icons:** Native SVG/Emojis

## 📂 Project Structure
Following standard React Architecture:
- `/src/components/` - Contains all reusable UI sections (Hero, About, Projects, etc.)
- `/src/index.css` - Global Tailwind CSS directives and dynamic Theme Variables mapping
- `/src/App.jsx` - Main application entry point that stitches components together

## 🛠️ Usage
1. Make sure you have Node.js installed.
2. Run `npm install` to install core dependencies.
3. Run `npm run dev` to start the local development server.
4. Run `npm run build` to create a production-ready optimized build output.
